Projects

Project-Tick

This a traditional tic-tac-toe game with the implementation of RESTful APi/CRUD, it allows user to create a account and then play a game of tic tac toe. The game is offline so user must take turns in each moves they make. Various technologies were used in this project; jQuery, Bootstrap, Javascript, HTML/CSS(SASS), along with working a API.

Landmarker

Landmarker is a fullstack web application that allows users to record various landmarks (ex. disney's world, Eiffel Tower ..etc..) that user's have been. Technologies used in this web application; Javascript, jQuery, Bootstrap, HTML/CSS(SASS), paired with a Express API(also built by me) (wunderlust-API) and deployed on heroku.

Bang It Out

This is a fullstack blogging web app with a public forum format that allows the user to join a community, speak one's mind, and get something off one's chest. Bang it Out lets users create a blog post and get connected with a community who thinks likewise.
